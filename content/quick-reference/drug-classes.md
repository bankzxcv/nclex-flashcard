# Drug Classes Quick Reference

## Cardiovascular Drugs

### ACE Inhibitors (-pril)
| Element | Details |
|---------|---------|
| **Prototype** | Lisinopril, Enalapril, Captopril |
| **Action** | Block conversion of angiotensin I to II; decrease BP & afterload |
| **Uses** | Hypertension, heart failure, post-MI, diabetic nephropathy |
| **Side Effects** | Dry cough (most common), hyperkalemia, angioedema, hypotension |
| **Nursing** | Monitor K+, BP; Hold if SBP <100; No salt substitutes; Take 1hr before meals |

### ARBs (-sartan)
| Element | Details |
|---------|---------|
| **Prototype** | Losartan, Valsartan, Irbesartan |
| **Action** | Block angiotensin II receptors; decrease BP |
| **Uses** | Hypertension, heart failure (if ACE not tolerated) |
| **Side Effects** | Hyperkalemia, hypotension, angioedema (rare), dizziness |
| **Nursing** | Same as ACE inhibitors but NO COUGH; Monitor K+ |

### Beta Blockers (-olol)
| Element | Details |
|---------|---------|
| **Prototype** | Metoprolol, Atenolol, Propranolol, Carvedilol |
| **Action** | Block beta receptors; decrease HR, BP, contractility |
| **Uses** | Hypertension, angina, MI, heart failure, arrhythmias |
| **Side Effects** | Bradycardia, hypotension, fatigue, bronchospasm, mask hypoglycemia |
| **Nursing** | Check HR before giving (hold if <60); Never stop abruptly; Caution in asthma/diabetes |

### Calcium Channel Blockers
| Element | Details |
|---------|---------|
| **Prototype** | Amlodipine (-dipine), Diltiazem, Verapamil |
| **Action** | Block Ca entry; vasodilation, decreased contractility/HR |
| **Uses** | Hypertension, angina, arrhythmias |
| **Side Effects** | Hypotension, peripheral edema, constipation (verapamil), bradycardia |
| **Nursing** | Monitor BP & HR; Avoid grapefruit juice; Take with food |

### Diuretics - Loop (Lasix)
| Element | Details |
|---------|---------|
| **Prototype** | Furosemide (Lasix), Bumetanide |
| **Action** | Block Na/K/Cl reabsorption in loop of Henle; potent diuresis |
| **Uses** | Heart failure, pulmonary edema, hypertension, renal failure |
| **Side Effects** | Hypokalemia, hypotension, ototoxicity, dehydration |
| **Nursing** | Monitor K+, I&O, daily weight; Give in AM; K+ supplementation needed |

### Diuretics - Thiazide (-thiazide)
| Element | Details |
|---------|---------|
| **Prototype** | Hydrochlorothiazide (HCTZ), Chlorothiazide |
| **Action** | Block Na reabsorption in distal tubule; moderate diuresis |
| **Uses** | Hypertension, mild heart failure |
| **Side Effects** | Hypokalemia, hyperglycemia, hypercalcemia, hyperuricemia |
| **Nursing** | Monitor K+, glucose, uric acid; Give in AM; Less potent than loop |

### Diuretics - Potassium-Sparing
| Element | Details |
|---------|---------|
| **Prototype** | Spironolactone (Aldactone), Triamterene |
| **Action** | Aldosterone antagonist; retain K+ |
| **Uses** | Heart failure, hypertension (with other diuretics) |
| **Side Effects** | HYPERKALEMIA, gynecomastia, irregular menses |
| **Nursing** | Monitor K+ closely; NO K+ supplements; NO salt substitutes |

### Digoxin (Lanoxin)
| Element | Details |
|---------|---------|
| **Prototype** | Digoxin |
| **Action** | Positive inotrope; negative chronotrope; increases contractility, decreases HR |
| **Uses** | Heart failure, atrial fibrillation |
| **Side Effects** | Bradycardia, arrhythmias, toxicity (N/V, yellow-green halos, confusion) |
| **Nursing** | Check apical pulse 60 sec (hold if <60); Therapeutic: 0.5-2.0 ng/mL; Hypokalemia increases toxicity |

### Nitrates
| Element | Details |
|---------|---------|
| **Prototype** | Nitroglycerin (SL, patch, IV), Isosorbide |
| **Action** | Vasodilation; decrease preload & afterload |
| **Uses** | Angina, heart failure, MI |
| **Side Effects** | Headache, hypotension, tachycardia, tolerance |
| **Nursing** | SL: up to 3 doses 5 min apart; Patch: rotate sites, remove at night; Monitor BP |

## Anticoagulants/Antiplatelets

### Warfarin (Coumadin)
| Element | Details |
|---------|---------|
| **Action** | Vitamin K antagonist; decreases clotting factors |
| **Uses** | DVT, PE, A-fib, artificial heart valves |
| **Monitoring** | PT/INR; Therapeutic INR 2-3 (mechanical valve 2.5-3.5) |
| **Antidote** | Vitamin K (phytonadione) |
| **Nursing** | Avoid vitamin K foods (greens); Bleeding precautions; Many drug interactions |

### Heparin
| Element | Details |
|---------|---------|
| **Action** | Activates antithrombin III; immediate anticoagulation |
| **Uses** | DVT, PE, MI, DIC (IV); DVT prophylaxis (SubQ) |
| **Monitoring** | aPTT (1.5-2x normal) |
| **Antidote** | Protamine sulfate |
| **Nursing** | SubQ in abdomen; Don't aspirate or massage; Monitor platelets (HIT risk) |

### Enoxaparin (Lovenox)
| Element | Details |
|---------|---------|
| **Action** | Low molecular weight heparin; anticoagulation |
| **Uses** | DVT prophylaxis, DVT/PE treatment |
| **Monitoring** | No aPTT needed |
| **Antidote** | Protamine sulfate (partial) |
| **Nursing** | SubQ in abdomen; Don't expel air bubble; Rotate sites |

### Antiplatelet - Aspirin
| Element | Details |
|---------|---------|
| **Action** | Inhibits platelet aggregation |
| **Uses** | MI prevention, stroke prevention, post-CABG |
| **Side Effects** | GI bleeding, tinnitus (toxicity) |
| **Nursing** | Take with food; Bleeding precautions; Stop 7 days before surgery |

### Antiplatelet - Clopidogrel (Plavix)
| Element | Details |
|---------|---------|
| **Action** | Inhibits platelet aggregation |
| **Uses** | MI, stroke, stent placement |
| **Side Effects** | Bleeding, TTP (rare) |
| **Nursing** | More potent than aspirin; Often given with aspirin (dual therapy) |

## Respiratory Drugs

### Beta2 Agonists - Short-Acting (SABA)
| Element | Details |
|---------|---------|
| **Prototype** | Albuterol (ProAir, Ventolin), Levalbuterol |
| **Action** | Bronchodilation via beta2 receptors |
| **Uses** | Acute asthma, bronchospasm, COPD exacerbation |
| **Side Effects** | Tachycardia, tremor, nervousness, hypokalemia |
| **Nursing** | Rescue inhaler; Use BEFORE other inhalers; Rinse mouth after |

### Beta2 Agonists - Long-Acting (LABA)
| Element | Details |
|---------|---------|
| **Prototype** | Salmeterol (Serevent), Formoterol |
| **Action** | Prolonged bronchodilation |
| **Uses** | Maintenance asthma/COPD (NOT acute) |
| **Side Effects** | Same as SABA but less intense |
| **Nursing** | NEVER for acute attack; Always with corticosteroid; BID dosing |

### Corticosteroids - Inhaled
| Element | Details |
|---------|---------|
| **Prototype** | Fluticasone (Flovent), Budesonide |
| **Action** | Anti-inflammatory; decrease airway inflammation |
| **Uses** | Maintenance asthma/COPD |
| **Side Effects** | Thrush, hoarseness, oral candidiasis |
| **Nursing** | Use AFTER bronchodilator; RINSE MOUTH after use; Takes 2-3 weeks for effect |

### Anticholinergics
| Element | Details |
|---------|---------|
| **Prototype** | Ipratropium (Atrovent), Tiotropium (Spiriva) |
| **Action** | Block acetylcholine; bronchodilation |
| **Uses** | COPD (first-line), asthma |
| **Side Effects** | Dry mouth, urinary retention |
| **Nursing** | Slower onset than albuterol; Can combine with albuterol (DuoNeb) |

## Diabetes Medications

### Insulin - Rapid-Acting
| Element | Details |
|---------|---------|
| **Types** | Lispro (Humalog), Aspart (NovoLog), Glulisine (Apidra) |
| **Onset/Peak/Duration** | 15 min / 1 hr / 3-4 hrs |
| **Uses** | Mealtime coverage, correction doses |
| **Nursing** | Give 15 min BEFORE meals; Clear appearance; Can give IV |

### Insulin - Short-Acting (Regular)
| Element | Details |
|---------|---------|
| **Types** | Regular insulin (Humulin R, Novolin R) |
| **Onset/Peak/Duration** | 30 min / 2-4 hrs / 6-8 hrs |
| **Uses** | Mealtime coverage, DKA (IV), sliding scale |
| **Nursing** | Give 30 min before meals; Clear; ONLY insulin for IV use |

### Insulin - Intermediate (NPH)
| Element | Details |
|---------|---------|
| **Types** | NPH (Humulin N, Novolin N) |
| **Onset/Peak/Duration** | 1-2 hrs / 4-12 hrs / 16-24 hrs |
| **Uses** | Basal coverage BID |
| **Nursing** | CLOUDY - roll gently; Give with meals; Peak = hypoglycemia risk |

### Insulin - Long-Acting
| Element | Details |
|---------|---------|
| **Types** | Glargine (Lantus), Detemir (Levemir), Degludec (Tresiba) |
| **Onset/Peak/Duration** | 1-2 hrs / NO peak / 24 hrs |
| **Uses** | Basal coverage once daily |
| **Nursing** | CLEAR; NO peak = less hypoglycemia; NEVER mix with other insulin; Any time of day |

### Metformin (Glucophage)
| Element | Details |
|---------|---------|
| **Action** | Decrease hepatic glucose production; increase insulin sensitivity |
| **Uses** | Type 2 diabetes (first-line oral) |
| **Side Effects** | GI upset, diarrhea, lactic acidosis (rare), vitamin B12 deficiency |
| **Nursing** | Take with food; Hold before contrast dye (48 hrs); Contraindicated in renal disease |

### Sulfonylureas (-ide)
| Element | Details |
|---------|---------|
| **Prototype** | Glyburide, Glipizide, Glimepiride |
| **Action** | Stimulate insulin release from pancreas |
| **Uses** | Type 2 diabetes |
| **Side Effects** | HYPOGLYCEMIA, weight gain |
| **Nursing** | Take 30 min before breakfast; Monitor glucose; Less used now |

## GI Medications

### Proton Pump Inhibitors (-prazole)
| Element | Details |
|---------|---------|
| **Prototype** | Omeprazole (Prilosec), Pantoprazole (Protonix), Esomeprazole (Nexium) |
| **Action** | Block H+/K+ ATPase pump; suppress gastric acid |
| **Uses** | GERD, PUD, H. pylori, stress ulcer prophylaxis |
| **Side Effects** | Headache, diarrhea, increased fracture risk, C. diff risk |
| **Nursing** | Take 30-60 min before breakfast; More effective than H2 blockers |

### H2 Receptor Blockers (-tidine)
| Element | Details |
|---------|---------|
| **Prototype** | Famotidine (Pepcid), Ranitidine (Zantac - recalled) |
| **Action** | Block histamine at H2 receptors; decrease acid |
| **Uses** | GERD, PUD, heartburn |
| **Side Effects** | Headache, confusion (elderly) |
| **Nursing** | Less potent than PPIs; Faster onset; Can give IV |

### Antiemetics - Ondansetron (Zofran)
| Element | Details |
|---------|---------|
| **Action** | 5-HT3 receptor antagonist; blocks serotonin |
| **Uses** | Chemotherapy N/V, post-op N/V |
| **Side Effects** | Headache, constipation, QT prolongation |
| **Nursing** | Very effective; Can give IV/PO/ODT; First-line for chemo N/V |

### Antiemetics - Metoclopramide (Reglan)
| Element | Details |
|---------|---------|
| **Action** | Dopamine antagonist; increase GI motility |
| **Uses** | N/V, GERD, gastroparesis |
| **Side Effects** | **Tardive dyskinesia** (irreversible), EPS, drowsiness |
| **Nursing** | Risk increases with >3 months use; Contraindicated in GI obstruction |

## Pain Management

### Opioids - Morphine
| Element | Details |
|---------|---------|
| **Action** | Mu receptor agonist; CNS depression, analgesia |
| **Uses** | Moderate-severe pain, MI pain, end-of-life care |
| **Side Effects** | Respiratory depression, constipation, sedation, N/V, hypotension |
| **Nursing** | Monitor respirations (hold if <12); Have naloxone available; Stool softeners needed |

### Opioids - Hydromorphone (Dilaudid)
| Element | Details |
|---------|---------|
| **Action** | Same as morphine but 5-7x more potent |
| **Uses** | Severe pain |
| **Side Effects** | Same as morphine |
| **Nursing** | High alert drug; Verify dose carefully; Same monitoring as morphine |

### Opioids - Fentanyl
| Element | Details |
|---------|---------|
| **Action** | Synthetic opioid; 50-100x more potent than morphine |
| **Uses** | Severe pain, anesthesia, chronic pain (patch) |
| **Side Effects** | Respiratory depression, rigidity (IV) |
| **Nursing** | Patch: takes 12-24 hrs to work; Fever increases absorption; Dispose properly |

### Opioid Antagonist - Naloxone (Narcan)
| Element | Details |
|---------|---------|
| **Action** | Reverses opioid effects at receptor sites |
| **Uses** | Opioid overdose, respiratory depression |
| **Onset** | IV: 1-2 min; IM/SubQ: 2-5 min; Intranasal: 2-5 min |
| **Nursing** | May need repeat doses; Monitor for withdrawal; Short half-life - watch for re-sedation |

### NSAIDs
| Element | Details |
|---------|---------|
| **Prototype** | Ibuprofen, Naproxen, Ketorolac (Toradol) |
| **Action** | Inhibit COX enzymes; decrease prostaglandins |
| **Uses** | Mild-moderate pain, fever, inflammation |
| **Side Effects** | GI bleeding, renal impairment, increased CV risk, bleeding |
| **Nursing** | Take with food; Avoid in renal disease; Short-term use only for ketorolac (5 days max) |

## Antibiotics

### Penicillins (-cillin)
| Element | Details |
|---------|---------|
| **Prototype** | Penicillin G/V, Amoxicillin, Ampicillin, Piperacillin |
| **Action** | Inhibit cell wall synthesis; bactericidal |
| **Uses** | Gram-positive infections, syphilis, strep throat |
| **Side Effects** | Allergic reactions (rash to anaphylaxis), diarrhea |
| **Nursing** | Ask about allergies; Cross-allergy with cephalosporins (10%); Take full course |

### Cephalosporins (Cef-)
| Element | Details |
|---------|---------|
| **Prototype** | Cefazolin, Ceftriaxone, Cefepime |
| **Action** | Inhibit cell wall synthesis; bactericidal |
| **Uses** | Broad-spectrum; surgical prophylaxis, pneumonia, UTI |
| **Side Effects** | Allergic reactions, diarrhea, C. diff |
| **Nursing** | 5 generations; Caution if PCN allergy; Take full course |

### Fluoroquinolones (-floxacin)
| Element | Details |
|---------|---------|
| **Prototype** | Ciprofloxacin (Cipro), Levofloxacin (Levaquin) |
| **Action** | Inhibit DNA synthesis; bactericidal |
| **Uses** | UTI, respiratory, anthrax |
| **Side Effects** | **Tendon rupture**, QT prolongation, phototoxicity, C. diff |
| **Nursing** | BLACK BOX warnings; Avoid in pregnancy/children; Avoid dairy/antacids (chelation) |

### Macrolides (-mycin/-thromycin)
| Element | Details |
|---------|---------|
| **Prototype** | Azithromycin (Z-pack), Erythromycin, Clarithromycin |
| **Action** | Inhibit protein synthesis; bacteriostatic |
| **Uses** | Respiratory infections, STIs, PCN alternative |
| **Side Effects** | GI upset, QT prolongation |
| **Nursing** | Z-pack: 5-day course; Take on empty stomach (except azithro) |

### Aminoglycosides (-micin/-mycin)
| Element | Details |
|---------|---------|
| **Prototype** | Gentamicin, Tobramycin, Amikacin |
| **Action** | Inhibit protein synthesis; bactericidal |
| **Uses** | Serious gram-negative infections |
| **Side Effects** | **Nephrotoxicity, Ototoxicity** (irreversible) |
| **Nursing** | Monitor: trough levels, BUN/Cr, hearing; IV only; Therapeutic drug monitoring required |

### Vancomycin
| Element | Details |
|---------|---------|
| **Action** | Inhibit cell wall synthesis |
| **Uses** | MRSA, C. diff (PO only), serious gram-positive |
| **Side Effects** | Red man syndrome (infusion reaction), nephrotoxicity, ototoxicity |
| **Nursing** | Infuse slowly (1-2 hrs); Monitor trough levels; Premedicate with antihistamine |

## Psych Medications

### SSRIs
| Element | Details |
|---------|---------|
| **Prototype** | Fluoxetine (Prozac), Sertraline (Zoloft), Escitalopram (Lexapro) |
| **Action** | Selective serotonin reuptake inhibitor |
| **Uses** | Depression, anxiety, OCD, PTSD |
| **Side Effects** | Sexual dysfunction, GI upset, weight gain, serotonin syndrome |
| **Nursing** | Takes 4-6 weeks for effect; Don't stop abruptly; Suicide risk initially |

### Benzodiazepines (-pam/-lam)
| Element | Details |
|---------|---------|
| **Prototype** | Lorazepam (Ativan), Diazepam (Valium), Alprazolam (Xanax) |
| **Action** | Enhance GABA; CNS depression |
| **Uses** | Anxiety, seizures, alcohol withdrawal, sedation |
| **Side Effects** | Sedation, dependence, respiratory depression, paradoxical reaction |
| **Nursing** | Risk of dependence; Taper to discontinue; Flumazenil = antidote |

### Antipsychotics - Typical (Haloperidol)
| Element | Details |
|---------|---------|
| **Action** | Block dopamine D2 receptors |
| **Uses** | Schizophrenia, acute psychosis, agitation |
| **Side Effects** | **EPS** (dystonia, akathisia, parkinsonism), tardive dyskinesia, NMS |
| **Nursing** | Monitor for EPS; Benztropine for EPS; NMS = medical emergency |

### Antipsychotics - Atypical
| Element | Details |
|---------|---------|
| **Prototype** | Risperidone, Olanzapine, Quetiapine, Clozapine |
| **Action** | Block dopamine & serotonin |
| **Uses** | Schizophrenia, bipolar disorder |
| **Side Effects** | Metabolic syndrome, weight gain, diabetes, less EPS than typical |
| **Nursing** | Clozapine: monitor WBC (agranulocytosis risk); Monitor glucose & lipids |

## Other Important Drug Classes

### Anticholinergics
| Element | Details |
|---------|---------|
| **Prototype** | Atropine, Benztropine, Diphenhydramine |
| **Effects** | Dry mouth, urinary retention, constipation, blurred vision, confusion |
| **Mnemonic** | "Blind as a bat, dry as a bone, red as a beet, mad as a hatter, hot as a hare" |

### Cholinergics
| Element | Details |
|---------|---------|
| **Prototype** | Bethanechol, Neostigmine, Pyridostigmine |
| **Effects** | SLUDGE: Salivation, Lacrimation, Urination, Defecation, GI upset, Emesis |
| **Uses** | Urinary retention, myasthenia gravis, reverse neuromuscular blockade |

### Corticosteroids (Systemic)
| Element | Details |
|---------|---------|
| **Prototype** | Prednisone, Methylprednisolone, Dexamethasone |
| **Uses** | Inflammation, autoimmune, allergic reactions, COPD exacerbation |
| **Side Effects** | Hyperglycemia, immunosuppression, osteoporosis, Cushing's syndrome |
| **Nursing** | Taper dose; Take in AM with food; Monitor glucose; Avoid live vaccines |
